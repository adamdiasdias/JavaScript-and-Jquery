Comparando um script com "receitas" "Manuais" "Guias"

Na receita, alguns scripts são simples e só lidam com um cenário individual, como uma receita simples. Outros scripts podem executar muitas tarefas para uma refeição complicada de três pratos.

No guias, grandes empresas constumm fornecer guias para novos funcionários que contêm procedimentos. Por exemplo, guias de hotéis para clientes fazer check-in, quando um quarto precisar ser arrumado, quando um alarme de incêndio dispara.

Nos manuais, mecânicos muitas vezes consultam manuais ao conserta automóveis. Esses manuais contêm uma séries de teste para verificar as principais funçoes dos carro. Da mesma forma, scripts podem permitir que o navegador verifique a situação atual e só siga um conjunto de passos..

Escrevendo um Script

Para desenvolver um Script é preciso indicar meu objetivo e listar as tarefas para alcançá-lo.

Um Script é simplemente uma série de instruçoes curtas.

Desenvolver uma idéia e separar por passos.

01 DEFINA O OBJETIVO - DEFINIR TAREFAS QUE VOCE DESEJA ALCANÇAR

02 PROJETE O SCRIOT - DIVIDE O OBJETIVO EM SERIES DE TAREFAS USANDO UM FLUXOGRAMA

03 CONDIFIQUE CADA PASSO - CADA UM DOS PASSOS DEVE SER ESCRITO EM UMA LINGUAGEM DE PROGRAMAÇAO QUE O COMPUTADOR ENTENDE. EM NOSSO CASO, É JAVASCRIPT.
